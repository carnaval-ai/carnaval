Real-World Scenarios

How frameworks behave in concrete enterprise situations.

📄

B2B Commercial Documents

Masking supplier order acknowledgements (OAs) or invoices while preserving purchase order numbers and product names for LLM processing.

MS Presidio: Fails to preserve context
LLM Masker: High latency & cost per run
Carnaval: Ideal (Preserves configs natively)
✉️

Multi-Lingual B2B Emails

Summarizing client exchange threads involving multiple languages (EN, FR, DE, JA) and B2B custom product references.

SpaCy NER: Requires multi-model setups
Scrubadub: Struggles with non-English entities
Carnaval: Ideal (GLiNER multi + 12 regex engines)
